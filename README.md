# Privacy Lens｜隐私政策阅读辅助工具

Privacy Lens 是一个面向互联网 App 隐私政策的轻量级阅读辅助工具。项目以电商平台和社交 App 为典型场景，将冗长、专业的隐私政策拆解为可理解、可检查、可输出的风险清单，帮助用户快速识别数据收集、第三方共享、个性化推荐、系统权限、社交关系链和用户权利等高关注问题。

## 项目背景

大多数用户在注册或使用 App 时，会直接点击"同意隐私政策"，但很少真正阅读其中的条款。隐私政策文本通常篇幅较长，专业性较强，普通用户难以判断哪些内容值得重点关注。

同时，在互联网法务和数据合规场景中，隐私政策审查也需要将复杂条款拆解为具体的产品功能、数据流转路径和用户权利问题。

因此，本项目尝试将隐私政策阅读过程工具化，帮助用户和合规初学者用更清晰的方式理解互联网 App 中常见的隐私风险。

## 核心功能

### 1. 六步隐私政策阅读指南

项目将隐私政策拆解为六个核心问题：

- 它收集了什么？
- 它为什么收集？
- 它会给谁？
- 它会怎么影响你？
- 它会存在哪里？
- 你能怎么控制？

这套框架适用于电商平台、社交 App、内容社区等常见互联网产品。

### 2. 隐私风险 Checklist

网站提供结构化 Checklist，帮助用户逐项检查隐私政策中的高关注风险，包括：

- 数据收集范围
- 使用目的与必要性
- 第三方共享与委托处理
- 个性化推荐与广告营销
- 跨境传输与数据存储
- 用户权利与退出机制
- 社交 App 专项风险

用户可以通过勾选清单，快速记录自己关注到的风险点。

### 3. 隐私概念词典

网站内置隐私概念词典，对常见概念进行用户友好化解释，例如：

- 个人信息
- 敏感个人信息
- 单独同意
- 第三方 SDK
- 个性化推荐
- 通讯录匹配
- 社交关系链
- 兴趣标签
- IP 属地

词典支持搜索和标签筛选，帮助用户快速理解隐私政策中的专业术语。

### 4. 真实案例拆解

案例区以电商平台和社交 App 为典型场景，拆解隐私政策中常见的数据处理问题。

案例包括：

- 注册登录与身份验证
- 下单交易与第三方共享
- 个性化推荐与商业短信
- 系统权限调用
- 第三方 SDK
- 用户权利与账号注销
- 通讯录好友推荐
- 位置与附近内容推荐
- 兴趣标签与算法推荐

每个案例按照"使用场景—涉及信息—核心问题—风险分析—用户应查看—用户可以怎么做"的结构展开。

### 5. 检查报告生成

用户完成 Checklist 勾选后，可以生成一份文本版隐私风险检查报告。报告会自动汇总：

- 检查时间
- 已勾选项目数量
- 高关注 / 中关注 / 低关注项目数量
- 重点风险项目
- 建议重点阅读的隐私政策部分
- 免责声明

该功能不依赖后端、数据库或 API，仅通过前端 JavaScript 实现。

## 设计思路

Privacy Lens 的核心设计思路是：
将复杂的法律文本转化为普通用户和产品同事能够理解的操作问题。

项目不是直接堆叠法律条文，而是围绕互联网 App 的真实产品场景展开，例如：

- 注册登录时，平台如何识别用户？
- 下单购物时，订单信息会流向哪些主体？
- 开启通讯录后，平台是否会进行好友推荐？
- 浏览、搜索、点赞、收藏是否会形成兴趣标签？
- 用户是否可以关闭个性化推荐、撤回授权或注销账号？

通过这种方式，项目把隐私政策阅读从"被动阅读长文本"转化为"主动检查关键风险"。

## 适用场景

### 普通用户

帮助普通用户快速理解隐私政策中的重点内容，尤其是系统权限、第三方共享、个性化推荐、营销触达、账号注销等常见问题。

### 法务 / 合规初学者

作为隐私政策审查训练模板，帮助建立结构化审查思路，理解不同数据处理场景下的合规关注点。

### 产品 / 运营人员

作为产品上线前的隐私风险初筛清单，用于检查新功能是否涉及新增数据收集、权限调用、第三方 SDK、营销推送或社交关系处理。

### 求职作品集

本项目也作为互联网法务、数据合规、隐私保护方向的个人作品，展示法律分析、产品理解、用户表达和 Legal-Tech 原型搭建能力。

## 项目价值

Privacy Lens 的实践价值不在于替代专业法律审查，而在于提供一个低门槛的隐私政策阅读框架。

它可以帮助用户从"直接点击同意"转向"知道应该重点关注什么"，也可以帮助法务和产品同事用更结构化的方式理解 App 数据处理流程。

从求职展示角度，本项目体现了以下能力：

- 数据合规分析能力
- 隐私政策结构化审查能力
- 法律文本用户友好化表达能力
- 互联网产品场景理解能力
- Legal-Tech 原型搭建能力
- 使用 AI / Vibe Coding 完成产品化实践的能力
